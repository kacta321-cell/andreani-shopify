Shopify  →  Andreani · Envío masivo

Despachá.

Subí el CSV de tus pedidos y descargá el Excel de Andreani listo para subir. Sin errores, sin instalar nada.

01 / EXPORTÁ
En Shopify: Pedidos → Exportar → CSV para Excel.
02 / SOLTÁ
Arrastrá ese CSV a la etiqueta de acá abajo.
03 / SUBÍ
Cargá el Excel en Andreani → Envío Masivo → A domicilio.
Despachado
🔒 Todo corre en tu navegador. Ningún pedido, dato de cliente ni dirección sale de tu computadora. No hay servidor: el archivo se arma localmente.
